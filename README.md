### 주요 기능
- Subway 꿀조합 인기순, 등으로 조회 / 등록 
- 써브웨이 꿀조합/메뉴 들의 솔직한 후기 
- 모든 메뉴들의 영양소, 가격 등의 정보 제공
- 이벤트 정보 
- 전국 써브웨이 매장 찾기 (근처 매장 찾기 / 매장명 검색 / 주소 검색)  
    
### 개발 언어
JAVA, Node.js, MySQL
